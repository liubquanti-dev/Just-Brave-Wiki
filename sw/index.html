Om server

Just Brave - pablic Terraria server av liubquanti.

Hur går man med?

Vår server stöder crossplay mellan mobila och stationära spelversioner.
För att ansluta till servern använder du adressen jbrave.liubquanti.click och port 7771.

Servern är inte uppe?

Kan du inte ansluta till servern? Kontrollera med övervakningssystemet att det fungerar.

Är den avstängd? Ta reda på varför i vårt uppdateringsflöde.
